HTML5 semantic tags

Semantic tags help the browser to understand where our content is all about. For example: If we place an article inside of a div tag, our browser has no idea that it is an article. Semantic tags aim to make our code more meaningful and solve this problem. Below I've listed a few semantic tags:

  • <main> </main> Contains the main content of our web page.
  • <acticle> </article> Contains an article.
  • <section> </section> Defines a section of the web page (contact info, blog, etc).
  • <aside> </aside> Related content which is not part of the main content.
  • <header> </header> For the header of the website (navigation bar, title, etc).
  • <footer> </footer> For the footer of the website.
