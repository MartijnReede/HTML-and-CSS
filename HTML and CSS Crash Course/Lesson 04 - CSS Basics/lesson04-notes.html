CSS (Style Sheets)

  • A style sheet is a list of CSS rules / rule sets.
  • Selectors are the HTML tags that we want to style in a CSS file.
  • A declaration is a line of code that gives a certain style to a selector.
  • If we use the <style> </style> tags or the style attribute, we can write CSS in our HTML file. A better way to do it, is to write an external CSS file.
  • If we would define a style in an HTML file, this needs to be done in between the <head> </head> tags.

How to use selectors and declarations

For example, if we want to style all div tags, we write something like this in the CSS file:

div {
color: red;
margin: 20px;
}

If we would like to style all list items, we would write something like this:

li{
font-weight:bold;
}

In the examples above, 'div' and 'li' are the selectors. "Color: red;" and "margin: 20px;" are declarations.

Using CSS inside of the HTML file

We can use the style attribute. For this paragraph I've used this code: <p style="color: green">

We can also use the style tags and place the code in between the head tags at the top of the HTML file. For this example, I've made all h3 tags orange using this code:

<style>
h3 {
color:orange;
}
</style>

Use CSS inside an external style sheet

  • Since we are using Thymeleaf, there is a specific place to store CSS style sheets. The place for this is: src/main/recourses --> static. Create a new folder named CSS there.
  • So how do we use a CSS style sheet? Between the head tags at the top of the HTML file, we use the following code: <link rel="stylesheet" href="locationOfFile"/>
  • So we use a link tag with the href=".." and a rel=".." attribute. Rel strands for relation. In this case, the relation is a stylesheet so we need to write: rel="stylesheet".

Inline and block elements

  • Inline elements don't take up any more room than their content needs.
  • Inline elements are span, img, strong, em, a and many more...
  • Block level elements take up the whole width of the page regardless of their content.
  • Block line elements are p, div, h2, h3, li and many more...

Reverse inline or block settings:

In short: Block elements take up the whole width of the page, Inline elements only take the space they need.

In CSS we can change a block element to inline or vice versa. Changing a div (block type) to inline:

div{
display: inline;
}

Changing a span (inline type) to block:

span{
display: block;
}

Margin and padding

  • Margin defines the space around the border of a particular element.
  • Padding defines the internal space between the content and the border.
  • To block level elements, padding and margin works as expected. To inline elements, marging and padding only works to the left and right.
