Some other useful tags

- <b> </b> or <strong> </strong> - Both are used to make text bold.

- <i> </i> or <em> </em> - Both are used to make text italicized.

- <small> </small> - Is used to make text slightly smaller.

- <a th:href="@{/lesson01}"> </a> - With thymeleaf, this is used to create a link back to lesson01: Go back!

- The style attribute can be used for CSS code. It can be placed inside another tag like this: <p style="color: red">Some red text</p>. This is red!

- Comments can be written in between these tags: <!-- some comment -->. Everything in between those tags will not be seen as HTML by the browser.

Lists

<ul> </ul> - Used as a wrapper of unordered lists.

  • <li>Is used for every item in the list.</li>
  • <li>Another item in the list.</li>
  • <li>A third item in the list.</li>

<ol> </ol> - Used as a wrapper of ordered lists. By default, it orders items by number.

  1. <li>Is used for every item in the list.</li>
  2. <li>Another item in the list.</li>
  3. <li>A third item in the list.</li>


Div and span

- <div> </div> - Div tags mark out different sections of the web page.

- Div tags will come in handy when using CSS.

- <span> </span> - Span tags also mark out different sections of the web page.

- Later in the course it will be explained how these can be used.


Single tags (not paired)

- Until now, all tags we used consist out of an opening and closing tag. But there are also single tags that we can use.

- <br/> - Is used to proceed
on the next line. It doesn't need a closing tag.

- <hr/> - Is used to create a horizontal line on the page.

- <img/> - Is used to show a picture on the web page, but we have to specify the location using src. Using Thymeleaf this would be: <img th:src="@{images/MarioImage.jpg}"/>

- To set the width or heigth of the picture we use width="..." heigth="...".

- The complete tag for the image below is: <img width="200" heigth="200" th:src="@{images/MarioImage.jpg}"/>

- The alt="" attribute is used to present an alternative in case the image cannot be found. Below I've used the following code: <img alt="Image can not be found!" th:src="@{NO/PICTURE/HERE}"/>
