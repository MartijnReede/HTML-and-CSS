Classes

  • Class attributes are used to point to specific selectors in CSS.
  • So if we want only one paragraph to have a specific color we can do: <p class="test"> Some text.. </p>.
  • In the CSS file, we're not using the p selector anymore. To refer to this particular paragraph we need to write: .test{some CSS code}.
  • Now this CSS code will apply to all the tags that have class="test". If we want the CSS code to only apply to p tags, we need to write: p.test{some CSS code} in the CSS file.
  • It is also possible to define two classes in a tag. This would look like: <p class="test example">. In CSS we can refer to it by: test.example{CSS code} or p.test.example{CSS code}

ID's

  • ID's can be used as classes but they are not so commonly used for CSS. More often they are used for JavaScript.
  • How can we use them in CSS? It's almost the same as with a class attribute: <p id="testID">
  • In the CSS file, we would refer to this id by: #testID{some CSS code}.
  • Just like classes, we can make the CSS code refer to only one specific tag type. If this would be a paragraph tag this would look like: p#testID{some CSS code}.

Parents and children

  • With using classes, we can be more specific about what elements we want to style.
  • For example: to style only paragraphs that are within a div tag, we use this CSS code: div p{some CSS code}.
  • Also it would be possible apply a style to certain classes that need to be in a div. In CSS this would look like: div .testclass{Some CSS code}.

Looking for attributes

  • In CSS we can also look for certain attributes. For example, if we want to style only anchor tags with an href inside, we use this CSS code: a[href]{some CSS code}.
  • If we want the attribute to have a certain value, we can refer to this in CSS by: a[href="www.test.com"]{some CSS code}.

Inheritance

Inheritance means that certain styles that are assigned to parent tags, will be inherited by their children. Mostly colors and font styles are inherited, there are also a lot of tags which are not inherited like borders or margin.

For example:

<div class="blue">
<p>Hello, I am blue...</p>
</div>

By using the CSS code from below, the text inside of the p tag is made blue because the style of the parent div is assigned to blue.

div.blue{
color: blue;
}
