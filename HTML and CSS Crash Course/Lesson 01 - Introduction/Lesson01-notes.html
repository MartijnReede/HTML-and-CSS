What is HTML?

- HTML stands for Hyper Text Markup Language.

- HTML structures content on a web page using tags.


What is CSS?

- CSS stands for Cascading Style Sheets.

- CSS is used to style our web pages to make them look better.

- We can change colors, position, effects, fonts, etc.


Some tags....

<!DOCTYPE html> - This says, look: this is an HTML file.

<html> </html> - This is used to surround everything else.

<head> </head> - In here we put some extra information about the web site like meta data or the title.

<body> </body> - This is all the stuff that we will see in the browser.

<p> </p> - This is a paragraph tag.


Inspect

To inspect a page, open Google Chrome, right click and select 'inspect'. With this functionality we see how any

web page is created. Also we can inspect our own code running on the particular page.
