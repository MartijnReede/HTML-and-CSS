Forms

  • Forms are basically multiple input fields. This lesson will show different types of them.
  • Everything that belongs to a form needs to be wrapped inside of form tags like this: <form> input fields </form>.
  • The id="someText" attribute can be used to link a certain style or label to input fields.
  • The placeholder="" attribute can be used place some light-gray text inside of the text fields.
  • Using the right input types is really important. If someone has to enter an email address and doesn't use an '@', the web page won't submit the form if the input type is email.
  • If we want to force the user to enter some data, we can add the required attribute. We don't have to give it any value. An example would be: <input type="text" required/>
  • For more input types check this link: W3 Schools.

The name attribute and radio buttons

  • The name attribute can be used to link different buttons together. If one button is checked, all other buttons become unchecked.
  • Radio buttons are a collection of buttons of which only one can be selected.
  • We use the value attribute to define what value stands for a selected button.

Example:

Select your gender:
Male
Female
Other

The code for the buttons above:

<input type="radio" value="Male" name="gender"/>Male
<input type="radio" value="Female" name="gender"/>Female
<input type="radio" value="Other" name="gender"/>Other

Text area

  • A text area is not created by input tags. It is made with different tags: <textarea> </textarea>.
  • We can set the size of the text area with the cols=".." and rows=".." attributes.

Example:


The code for the text area above:

<label for="textArea">Enter a description:</label>
<textarea id="textArea" cols="30" rows="10" placeholder="Your description here please..."></textarea>

The submit button

  • If a submit button is placed between form tags, user input will be submitted via a post request.
  • The value attribute in this case, stands for the text inside of the button.

Example:

The code for the button above:

<input type="submit" value="!! SUBMIT THE FORM !!">
